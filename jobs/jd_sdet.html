Software Development Engineer in Test (SDET)


Apply
  • overview

  • We’re looking for a mid-level Software Development Engineer in Test with around five years of experience who can own the entire automated-testing pipeline for our Unity titles. You will design and code NUnit-based tests, extend the Unity Test Framework, and build the CI infrastructure that runs those tests across PC, Android and iOS builds. In our small, close-knit team you’ll collaborate with gameplay, tools and DevOps engineers to deliver rock-solid code quality at speed.

  • responsibilities
    • Design, develop and maintain unit, integration and end-to-end tests for Unity projects using the Unity Test Framework and NUnit.
    • Build, own and continuously improve the automation framework & test harnesses; integrate them into CI pipelines (Jenkins / GitHub Actions / Azure DevOps).
    • Provision and monitor the test-execution infrastructure – build triggers, parallel runners, dashboards and coverage reports.
    • Partner with gameplay and tools engineers to design testable architecture, inject mocks/stubs and shift quality checks left.
    • Analyse flaky tests and CI failures; drive root-cause fixes or framework improvements.
    • Track and report quality metrics (pass-rate, coverage, MTTR); champion defect-prevention initiatives.
    • Conduct code reviews for test and production code; mentor junior engineers on test-automation best practices.

  • candidate profile
    • Bachelor’s degree (or equivalent) in Computer Science, Software Engineering or related field.
    • 5+ years writing automated tests and frameworks in C#/.NET – with at least 3 years in Unity projects.
    • Deep knowledge of NUnit, the Unity Test Framework and test-driven development.
    • Proven experience building and maintaining CI/CD pipelines for multi-platform game builds (Jenkins, GitHub Actions, Azure).
    • Solid understanding of test methodologies (unit, component, integration, smoke, performance) and the test-pyramid philosophy.
    • Comfortable with Git, branching strategies and code-review workflows.
    • Excellent debugging, analytical and communication skills; able to translate failures into actionable engineering tasks.
    • Passion for games, quality craftsmanship and collaborative problem-solving.
    Plusses:
    • Exposure to cloud device farms (AWS Device Farm, Firebase Test Lab) or containerised test runners (Docker, Kubernetes).
    • Knowledge of performance / load-test tooling for Unity (Profiler, dotTrace, etc.).
    • Familiarity with mobile build pipelines (Gradle, Xcode) and store-submission checks.
