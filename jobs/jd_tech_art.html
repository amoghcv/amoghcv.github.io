Technical artist (UI & programming)


Apply
  • overview

  • We're looking for an experienced Technical Artist (UI & Programming) who wants to work on our Mobile games. This role sits at the intersection of UI, art and engineering, with a strong emphasis on programming-driven solutions. You will be responsible for building scalable UI pipelines, implementing performant UI systems and creating tools that enable artists and designers to work efficiently while maintaining visual quality.

  • responsibilities
    • Design, implement and maintain robust UI pipelines and runtime UI systems.
    • Work closely with designers and artists to translate UI/UX designs into performant in-game UI.
    • Develop tools, scripts and editor extensions to automate UI and art workflows.
    • Profile, optimize and debug UI performance, memory usage and draw calls on mobile devices.
    • Implement UI animations, transitions and effects with a strong focus on responsiveness and polish.
    • Collaborate with engineers to ensure UI systems integrate cleanly with gameplay systems.
    • Establish best practices for UI architecture, asset organization and technical constraints.
    • Mentor artists and designers on technical considerations related to UI implementation.

  • candidate profile
    • A minimum of 5+ years' experience as a Technical Artist with strong UI and programming focus.
    • Excellent proficiency in C# (Unity) and experience building tools, editors and runtime systems.
    • Strong understanding of UI frameworks, layout systems, animation and state-driven UI.
    • Hands-on experience optimizing UI for mobile performance and multiple aspect ratios.
    • Ability to work comfortably across art, design and engineering disciplines.
    • Strong debugging, profiling and problem-solving skills.
    • Experience creating reusable UI components and scalable UI architectures.
    • Self-driven, detail-oriented and comfortable owning complex technical systems.
    • Passion for games and high-quality player experiences.
    Plusses:
    • Experience shipping UI-heavy or live service mobile games.
